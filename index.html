Welcome!

Hey! This is the support page for my HSC Major Work, a desktop application I wrote for Windows PC's that can generate a maze and then find the most efficient route to the solution. It features Depth First Search, and A*. If you'd like to play around with it please use the online installer, if you wanna check out the source code it's stored here, and if you want to see the documentation that's available too!
